# weatherforecast_imd
Scraps the weather forecasting data from IMD website.



### How to use:
_usage_: main.py [-h] [-c CITY] [-l] [-f]

Fetch weather information from IMD

optional arguments:
  -h, --help            show this help message and exit
  -c CITY, --city CITY
  -l, --list
  -f, --forecast
